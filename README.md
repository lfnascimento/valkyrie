# Valkyrie - Best Available Seats

To run this project you need to have:

- **Docker**

## Setup the project locally
```
$ git clone https://github.com/lfnascimento/valkyrie.git
$ cd valkyrie
$ docker-compose build
```

## Running the project

```$ docker-compose up```
